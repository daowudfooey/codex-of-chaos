# Codex of Chaos

Codex of Chaos is a harmless, single-page parody interface inspired by theatrical death-metal artwork. It is fictional, is not affiliated with ChatGPT or any real authentication service, and does not perform authentication.

## Run it

Open `index.html` directly in a modern browser. No server, build step, dependencies, fonts, images, or network connection are required.

## Privacy and safety

- The fake login has no external form action.
- Its fields have no `name` attributes and their contents are never read by JavaScript.
- Submitting only prevents the browser's default form behavior, clears the fields, and reveals a fictional rejection message.
- The page uses no `fetch`, `XMLHttpRequest`, `WebSocket`, redirects, cookies, local storage, analytics, or tracking.
- All styling, scripts, and SVG artwork are embedded in `index.html`.

This project is visual satire only. Do not modify it to collect real credentials or impersonate a real service.
